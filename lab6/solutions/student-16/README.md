# Лабораторная работа №6 — Библиотека для работы с текстовыми файлами

**Студент:** 16  
**Вариант:** 4 — Анализ текстового файла

## Описание

Библиотека `txtfile` реализует анализ текстового файла:

- `read_file` — считывает файл и возвращает массив строк (NULL-terminated)
- `free_lines` — освобождает память массива строк
- `print_lines` — выводит строки в stdout
- `count_lines` — возвращает количество строк
- `count_words` — возвращает количество слов
- `count_chars` — возвращает суммарное количество символов
- `longest_line_length` — возвращает длину самой длинной строки

## Сборка и запуск

```bash
make
./analyze_file data/input.txt
```

## Пример вывода

```
Lines:   5
Words:   18
Chars:   82
Longest: 26
```
